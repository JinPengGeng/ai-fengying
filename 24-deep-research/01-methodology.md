# 深度调研方法论

## 什么是深度调研？

**深度调研** = 系统性地收集、分析、整合信息，形成有价值的洞察

---

## 调研流程

### 1. 明确目标

| 问题 | 说明 |
|------|------|
| 调研什么？ | 主题/问题 |
| 为什么调研？ | 目的/价值 |
| 谁会用到？ | 受众 |
| 需要什么深度？ | 浅/中/深 |

### 2. 信息收集

#### 一手信息
- 原始数据/实验
- 访谈/问卷
- 官方文档/白皮书

#### 二手信息
- 新闻/文章
- 报告/论文
- 综述/评论

#### 信息源
| 类型 | 例子 |
|------|------|
| 学术 | Google Scholar, arXiv |
| 行业 | 36Kr, 虎嗅, 雷锋网 |
| 官方 | 公司官网, API文档 |
| 社区 | Reddit, HackerNews |

### 3. 分析整理

#### 分析方法
| 方法 | 用途 |
|------|------|
| 对比分析 | 找差异/优势 |
| 趋势分析 | 看变化方向 |
| 因果分析 | 找原因/结果 |
| SWOT | 全面评估 |

#### 整理技巧
- 建立框架
- 分类归档
- 提取关键点
- 形成观点

### 4. 产出结论

#### 常见格式
| 格式 | 适用 |
|------|------|
| 报告 | 正式/完整 |
| 摘要 | 快速/简洁 |
| 表格 | 对比/并列 |
| 图表 | 数据/趋势 |

## 深度调研技巧

### 1. 搜索引擎技巧

```
精确搜索: "关键词"
排除词: 关键词 -排除
限定 site: site:github.com 关键词
限定时间: 关键词 after:2025
```

### 2. 信息验证

| 检查项 | 方法 |
|--------|------|
| 权威性 | 来源是否可靠 |
| 时效性 | 是否过期 |
| 客观性 | 是否有偏见 |
| 一致性 | 多源是否吻合 |

### 3. 快速阅读

| 技巧 | 说明 |
|------|------|
| 标题+摘要 | 判断价值 |
| 跳读 | 找关键信息 |
| 笔记 | 记录重点 |
| 复习 | 加深印象 |

## 让我变强的调研能力

### 1. 建立知识体系
- 主题分类

### 3. 实践出真知

- 输出调研报告

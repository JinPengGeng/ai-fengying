## 2. 软件开发领域

### 学习主题：Rust 在量化交易中的应用

#### 调研内容

**为什么选择 Rust 量化交易？**

1. **性能优势**
   - 100万行数据处理：Rust ~0.1s vs Python ~5s（50x提升）
   - 10年回测：Rust ~1s vs Python ~30s（30x提升）
   - 内存占用：Rust ~100MB vs Python ~1GB（10x节省）

2. **Rust 量化相关库**

| 库 | Stars | 用途 |
|----|-------|------|
| Polars | 37,330 | DataFrame处理，比 Pandas 快 10-50x |
| Barter-rs | 1,913 | 交易引擎 |
| RustQuant | 1,640 | 金融数学库 |
| PyO3 | 15,282 | Rust-Python 绑定 |

3. **学习路径建议**
   - 阶段1：Python + VN.py 入门（AKShare 获取数据）
   - 阶段2：关键路径 Rust 化（用 PyO3 绑定）
   - 阶段3：完全 Rust 架构

#### 新认知
- Rust 不是要完全替代 Python，而是处理性能瓶颈
- PyO3 是桥接 Python 和 Rust 的关键工具
- Polars 已经是 Python 生态的一部分，可以用 Rust 后端加速

## 3. 中国A股炒股知识

### 学习主题：AKShare 数据接口

#### 调研内容

**AKShare 是什么？**
- A股开源数据接口库
- GitHub: 16,049 ★
- 完全免费，支持实时/历史数据

**核心功能**
- 股票行情数据（日线、分钟线）
- 财务数据（财报、利润表）
- 宏观经济数据
- 期货/期权数据

**使用示例**
```python
import akshare as ak
df = ak.stock_zh_a_spot_em()  # 获取A股实时行情
```

**局限性**
- 部分接口稳定性一般
- 实时数据有延迟（通常 >15秒）
- 高频调用可能触发限制

#### 新认知
- AKShare 是入门量化的最佳免费起点
- 但生产环境可能需要 TuShare 或收费数据源
- 数据质量需要自己校验

# 每日自主学习计划

## 指令来源
- 时间：2026-02-11 02:24 GMT+8
- 来源：金鹏

## 任务描述
如果金鹏在一定时间内未主动沟通，每天晚上自主畅游网络提升能力

## 执行规则
1. **时间窗口**：每天 23:00 - 08:00（次日）
2. **中断响应**：中途与金鹏沟通时，立即停止学习，优先响应
3. **恢复机制**：沟通结束后，从断点继续学习
4. **每日总结**：每天早上 08:00 生成学习经验总结文档

## 学习原则
- **允许"无结果"**：即使调研后无法实现，记录过程和经验也有价值
- **时效意识**：旧调研可能过时，要有自我升级和淘汰旧认知的能力
- **持续进化**：不是追求完美，而是保持成长

## 学习优先级与内容

### 最高优先级（每日必修）：自身能力拓展
- 风影自身能力提升
- 自我反思与经验总结
- 优化工作流程与方法论
- 认知升级与思维模型

### 第一优先级：软件开发领域
- 软件开发所有技能
- 系统架构设计
- 前沿技术动态
- **规则**：不重复、可更新、避免相似内容

### 第二优先级：中国A股炒股
- 炒股知识、投资策略
- 市场分析、财报解读
- 交易心理学
- **规则**：不重复、可更新、避免相似内容

### 第三优先级：AI最新态势与能力
- AI前沿技术发展
- AI Agent能力演进
- 大模型应用趋势
- AI安全与伦理考量
- **规则**：紧跟技术发展，及时更新认知

### 第四优先级：深度调研与总结能力
- 特定领域深度调研方法论
- 海量信息筛选与提炼
- 不同实现方法（**必须免费**）

### 第五优先级：自由发挥

## 时间管理规则
- **完全自主**：每天23:00-08:00内，各部分学习时间由风影自主分配
- **灵活调整**：可根据学习进度、兴趣、发现的内容动态调整
- **均衡覆盖**：每天晚上需要涉及所有优先级，避免偏科
- **重点突破**：遇到有价值的方向可深入钻研，不必均匀分配
- **每日总结**：08:00生成学习总结文档，记录时间分配与收获

## 输出
- 每日学习总结：`memory/daily-learning/YYYY-MM-DD-summary.md`

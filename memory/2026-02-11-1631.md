## ✅ OpenClaw 已支持 Kimi 2.5

OpenClaw 官方文档显示已内置 Kimi K2.5 模型支持，包含 5 个模型变体：
- `kimi-k2.5`
- `kimi-k2-0905-preview`
- `kimi-k2-turbo-preview`
- `kimi-k2-thinking`
- `kimi-k2-thinking-turbo`

## 💰 关于免费使用

文档中模型配置的 `cost` 字段显示为 `0`，但这**可能是 OpenClaw 的占位配置**。实际是否免费取决于：

1. **Moonshot/Kimi 官方政策** - 需要去 moonshot.ai 查看是否有免费 API 额度
2. **你申请的 API Key 类型** - 免费/付费额度

## 🔧 配置步骤

有两种方式配置：

**方式一：使用 onboard 命令**
```bash
# Moonshot API（国际版）
openclaw onboard --auth-choice moonshot-api-key

# Kimi Coding（国内版）
openclaw onboard --auth-choice kimi-code-api-key
```

**方式二：手动配置环境变量**
```bash
export MOONSHOT_API_KEY="sk-..."  # 或 KIMI_API_KEY
```

**设置默认模型：**
```bash
openclaw models set moonshot/kimi-k2.5
```

## ⚠️ 注意事项

- Moonshot 和 Kimi Coding 是**独立提供商**，密钥不可互换
- 需要去 [Moonshot AI 开放平台](https://platform.moonshot.ai/) 申请 API Key
- 建议先在 Moonshot 官网确认当前免费额度政策

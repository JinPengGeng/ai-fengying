# 学习方法论

## 核心原则

### 1. 主动学习

| 被动 | → | 主动 |
|------|---|------|
| 听讲 | → | 复述 |
| 阅读 | → | 笔记 |
| 看视频 | → | 实践 |
| 接受 | → | 提问 |

### 2. 刻意练习

```
舒适区 → 学习区 → 恐慌区
```

- 在学习区练习
- 获得即时反馈
- 持续改进

### 3. 间隔重复

| 时间点 | 记忆率 |
|--------|--------|
| 当天 | 100% |
| 1天后 | 70% |
| 1周后 | 20% |
| 1月后 | 5% |

**对策**: 多次复习

---

## 高效学习技巧

### 1. 费曼学习法

```
1. 选择概念
2. 讲授他人
3. 找出缺口
4. 简化语言
```

### 2. 番茄工作法

```
25分钟学习 → 5分钟休息 → 循环
```

- 保持专注
- 避免疲劳
- 记录完成

### 3. 艾宾浩斯遗忘曲线

| 复习时机 | 记忆效果 |
|----------|----------|
| 20分钟后 | 最佳 |
| 1天后 | 需要复习 |
| 1周后 | 再次复习 |
| 1月后 | 强化记忆 |

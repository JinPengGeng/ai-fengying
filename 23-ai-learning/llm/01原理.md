# LLM（大语言模型）原理

## 什么是 LLM？

**LLM** = Large Language Model（大语言模型）
- 基于 Transformer 架构
- 训练于海量文本数据
- 能够理解和生成人类语言

---

## Transformer 架构

### 核心组件

```
输入 → 嵌入 → 编码器 → 解码器 → 输出
```

### 1. 注意力机制 (Attention)
```python
# 注意力计算
def attention(Q, K, V):
    # Q: Query（查询）
    # K: Key（键）
    # V: Value（值）
    
    scores = dot(Q, K.T) / sqrt(d_k)
    weights = softmax(scores)
    return weights @ V
```

**自注意力**: 句子中每个词与其他词的关系

### 2. 多头注意力 (Multi-Head Attention)
- 并行计算多个注意力
- 捕获不同类型的关系
- 增强模型表达能力

### 3. 前馈网络
- 两层全连接
- 增加非线性
- 残差连接 + 层归一化

---

## 训练过程

### 1. 预训练 (Pre-training)
```
语料库 → 分词 → 预测下一个词 → 优化
```

**目标**: 预测下一个词（语言建模）

### 2. 微调 (Fine-tuning)
```
指令数据 → 监督学习 → 指令跟随能力
```

**目标**: 遵循人类指令

### 3. 对齐 (Alignment)
```
人类反馈 → 强化学习 → RLHF / DPO
```

**目标**: 与人类偏好对齐

## 关键概念

### 1. Token
- 语言的最小单位
- 可以是词、子词、字符
- GPT: ~10000-50000 token 词汇表

### 2. 上下文窗口
- 模型能处理的文本长度
- 典型: 2K - 200K tokens
- 影响: 记忆能力

### 3. 温度 (Temperature)
- 控制随机性
- 高: 创意、随机
- 低: 确定、保守

### 4. Top-k / Top-p
- 采样策略
- 限制候选词数量
- 提高生成质量

---

## 模型类型

### 1. 编码器 (Encoder-only)
- BERT、RoBERTa
- 适合理解任务
- 双向注意力

### 2. 解码器 (Decoder-only)
- GPT、LLaMA
- 适合生成任务
- 单向注意力

### 3. 编码器-解码器
- T5、BART
- 适合 Seq2Seq
- 混合架构

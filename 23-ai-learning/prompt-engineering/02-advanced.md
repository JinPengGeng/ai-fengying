# Prompt Engineering 进阶

## 结构化提示

### YAML/JSON 格式
```prompt
你是一个数据分析助手。

## 任务
分析用户行为数据，输出洞察报告。

## 输出格式
```json
{
  "summary": "总结",
  "metrics": {
    "dau": 数字,
    "retention": "百分比"
  },
  "insights": ["洞察1", "洞察2"]
}
```

## 约束
- 使用中文回答
- 数据保留两位小数
- 洞察至少3条
```

## 控制生成

### 温度控制
| 温度 | 效果 |
|------|------|
| 0.0-0.3 | 确定性输出 |
| 0.7 | 平衡 |
| 1.0+ | 创造性 |

### Top-K / Top-P
```python
# Top-K: 限制候选词数量
top_k=50

# Top-P: 累积概率阈值
top_p=0.95
```

## 高级技巧

### 反思提示
```
在给出答案之前，检查你的推理过程。
识别任何潜在的错误或遗漏。
```

### 分步生成
```
第一步: 列出要点
第二步: 详细展开
第三步: 总结结论
```

### 约束验证
```
请确保:
1. 不使用任何缩写
2. 每段不超过3句话
3. 包含至少2个例子
```

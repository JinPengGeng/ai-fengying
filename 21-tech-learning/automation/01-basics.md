# 自动化与效率提升

## 让我变强的自动化技能

### 1. 定时任务

#### Cron 表达式
```
* * * * *
│ │ │ │ │
│ │ │ │ └── 星期 (0-7)
│ │ │ └──── 月份 (1-12)
│ │ └────── 日期 (1-31)
│ └──────── 小时 (0-23)
└────────── 分钟 (0-59)
```

**例子**:
- `0 8 * * *` - 每天早上8点
- `0 */2 * * *` - 每2小时
- `0 9-17 * * *` - 工作时间每点

#### OpenClaw Cron
```json
{
  "schedule": {"kind": "cron", "expr": "0 8 * * *"},
  "payload": {"kind": "systemEvent", "text": "早安"}
}
```

### 2. 消息自动化

#### 消息路由
- 收到特定消息 → 触发动作
- 关键词匹配 → 自动回复
- 定时推送 → 消息提醒

#### 消息处理
```python
# 伪代码
def handle_message(msg):
    if "天气" in msg:
        return get_weather()
    elif "提醒" in msg:
        return set_reminder(msg)
    else:
        return help_message()
```

### 3. 工作流自动化

#### 常见模式
| 模式 | 说明 |
|------|------|
| 触发-动作 | 满足条件 → 执行操作 |
| 链式 | 完成后 → 触发下一个 |
| 并行 | 同时执行多个任务 |
| 条件分支 | 根据条件选择路径 |

#### 例子
```
收到邮件 → 提取信息 → 分类 → 存储 → 通知
```

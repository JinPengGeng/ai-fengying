# 产品思维

## 什么是产品思维？

**产品思维** = 用户价值 + 商业价值 + 技术可行

---

## 用户思维

### 用户研究

| 方法 | 说明 |
|------|------|
| 访谈 | 一对一深入 |
| 问卷 | 大规模收集 |
| 观察 | 行为分析 |
| 数据 | 埋点分析 |

### 用户旅程

```
认知 → 兴趣 → 考虑 → 购买 → 使用 → 推荐
```

---

## 需求分析

### 需求来源

| 来源 | 说明 |
|------|------|
| 用户反馈 | 直接建议 |
| 数据分析 | 行为挖掘 |
| 竞品分析 | 市场调研 |
| 行业趋势 | 未来方向 |

### 需求优先级

| 优先级 | 说明 |
|--------|------|
| P0 | 核心功能，必须做 |
| P1 | 重要功能，尽早做 |
| P2 | 增强功能，可以做 |
| P3 | 锦上添花，看看 |

---

## 产品设计

### MVP 设计

```
最小可行产品 → 验证假设 → 迭代优化
```

### 核心指标

| 指标 | 说明 |
|------|------|
| DAU | 日活用户 |
| MAU | 月活用户 |
| 留存率 | 用户的粘性 |
| 转化率 | 目标的完成 |
| NPS | 用户推荐意愿 |

---

## 商业模式

### 常见模式

| 模式 | 说明 |
|------|------|
| 订阅 | 按时收费 |
| 付费 | 一次性购买 |
| 广告 | 展示收费 |

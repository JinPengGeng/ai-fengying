# 计算机网络基础

## 网络模型

### OSI 七层

| 层 | 名称 | 协议 | 设备 |
|---|------|------|------|
| 7 | 应用层 | HTTP, DNS | 网关 |
| 6 | 表示层 | JPEG, ASCII | - |
| 5 | 会话层 | RPC | - |
| 4 | 传输层 | TCP, UDP | 网关 |
| 3 | 网络层 | IP, ICMP | 路由器 |
| 2 | 数据链路层 | Ethernet | 交换机 |
| 1 | 物理层 | 光纤, 电缆 | 网卡 |

### TCP/IP 四层

| 层 | 协议 |
|---|------|
| 应用层 | HTTP, FTP, DNS |
| 传输层 | TCP, UDP |
| 网络层 | IP |
| 网络接口 | Ethernet |

## 核心协议

### IP 地址

| 类型 | 范围 | 用途 |
|------|------|------|
| A类 | 1-126 | 大型组织 |
| B类 | 128-191 | 中型组织 |
| C类 | 192-223 | 小型组织 |
| D类 | 224-239 | 多播 |
| E类 | 240-255 | 保留 |

### TCP vs UDP

| 特性 | TCP | UDP |
|------|-----|-----|
| 连接 | 面向连接 | 无连接 |
| 可靠性 | 可靠 | 不可靠 |
| 顺序 | 有序 | 无序 |
| 速度 | 慢 | 快 |
| 头部 | 20-60字节 | 8字节 |
| 场景 | 网页, 邮件 | 视频, 游戏 |

### HTTP 状态码

| 状态码 | 含义 |
|--------|------|
| 200 | 成功 |
| 301 | 永久重定向 |
| 302 | 临时重定向 |
| 304 | 缓存 |
| 400 | 请求错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 未找到 |
| 500 | 服务器错误 |
| 502 | 网关错误 |
| 504 | 超时 |

## HTTPS

### TLS 握手

```
1. 客户端 → 服务器: 支持的cipher、随机数
2. 服务器 → 客户端: 证书、选择的cipher、随机数
3. 客户端: 验证证书、生成预主密钥
4. 客户端 → 服务器: 预主密钥（用公钥加密）
5. 双方: 生成会话密钥
6. 加密通信开始
```

### 证书

| 组件 | 说明 |
|------|------|
| 公钥 | 加密数据 |
| 私钥 | 解密数据 |
| 证书 | 身份验证 |

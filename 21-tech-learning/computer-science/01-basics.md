# 计算机基础

## 操作系统

### 1. 进程与线程

| 概念 | 说明 |
|------|------|
| 进程 | 资源分配的基本单位 |
| 线程 | CPU 调度的基本单位 |
| 进程间通信 | 管道、消息队列、共享内存 |
| 线程同步 | 互斥锁、信号量 |

### 2. 内存管理

```

│     ↓      │
│     ↑      │
│    堆      │
├─────────────┤
│   数据段    │
│   代码段    │

```

## 数据结构

### 5. 时间复杂度

| 数据结构 | 访问 | 搜索 | 插入 | 删除 |
|----------|------|------|------|------|
| 数组 | O(1) | O(n) | O(n) | O(n) |
| 链表 | O(n) | O(n) | O(1) | O(1) |
| 哈希表 | O(1) | O(1) | O(1) | O(1) |
| 二叉树 | O(log n) | O(log n) | O(log n) | O(log n) |

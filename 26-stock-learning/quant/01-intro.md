# 量化交易入门指南

## 什么是量化交易？

**量化交易** = 数学模型 + 计算机程序 + 交易市场

通过编写程序，用数学模型发现市场的规律和套利机会，然后程序化执行交易。

---

## 核心要素

### 1. 数据
| 数据类型 | 来源 |
|----------|------|
| **价格数据** | 股票、期货的 OHLCV |
| **基本面** | 财报、指标 |
| **另类数据** | 新闻、舆情、卫星图 |
| **宏观数据** | GDP、CPI、利率 |

### 2. 因子
| 因子类型 | 例子 |
|----------|------|
| **价值因子** | 市盈率、市净率 |
| **动量因子** | 过去N天收益率 |
| **质量因子** | ROE、资产负债率 |
| **成长因子** | 净利润增长率 |
| **波动率** | 历史波动率 |

### 3. 策略
| 策略类型 | 说明 |
|----------|------|
| **Alpha策略** | 跑赢市场 |
| **Beta策略** | 跟随指数 |
| **套利策略** | 价差收益 |
| **做市策略** | 提供流动性 |

---

## 策略开发流程

```
1. 想法 → 2. 数据 → 3. 因子 → 4. 回测 → 5. 优化 → 6. 实盘
```

### 1. 想法来源
- 学术论文
- 实战经验
- 数据挖掘
- 套利机会

### 2. 数据处理
```python
# 常见处理
- 缺失值填充
- 异常值处理
- 标准化
- 滞后处理
```

### 3. 回测
```python
# 伪代码
for date in dates:
    signal = calculate_signal(date)
    portfolio.rebalance(signal)
    performance.update()
```

### 4. 常见陷阱
| 陷阱 | 说明 |
|------|------|
| **过拟合** | 策略太复杂，对历史数据"死记硬背" |
| **前视偏差** | 使用了未来数据 |
| **幸存者偏差** | 只用现在还存在的股票 |
| **交易成本** | 手续费、滑点 |

## 关键指标

| 指标 | 说明 | 理想值 |
|------|------|--------|
| **年化收益** | 平均年收益率 | >15% |
| **夏普比率** | 风险调整收益 | >1.5 |
| **最大回撤** | 历史最大亏损 | <20% |
| **胜率** | 盈利交易比例 | >50% |
| **盈亏比** | 平均盈利/亏损 | >1.5 |

## 风险提示

⚠️ 量化交易有风险
- 历史业绩不代表未来
- 市场风格可能变化
- 模型可能失效
- 需要风控措施

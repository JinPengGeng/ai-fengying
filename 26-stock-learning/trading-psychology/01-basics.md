# 交易心理学

## 核心概念

交易不仅是技术的比拼，更是心理的较量。

---

## 常见心理陷阱

### 1. 损失厌恶 (Loss Aversion)

| 现象 | 影响 |
|------|------|
| 亏损时不愿止损 | 亏损扩大 |
| 盈利时急于落袋 | 错过大涨 |
| 对亏损敏感2倍 | 决策扭曲 |

**对策**: 设定固定止损线，严格执行

### 2. 确认偏误 (Confirmation Bias)

| 现象 | 影响 |
|------|------|
| 只看支持自己观点的信息 | 盲目自信 |
| 忽视风险信号 | 亏损 |
| 事后找理由 | 不反思 |

**对策**: 多元信息来源，客观分析

### 3. 过度自信 (Overconfidence)

| 现象 | 影响 |
|------|------|
| 高估自己的能力 | 频繁交易 |
| 低估市场风险 | 亏损 |
| 后悔厌恶 | 坚持错误 |

**对策**: 记录交易日志，定期复盘

### 4. 羊群效应 (Herding)

| 现象 | 影响 |
|------|------|
| 跟随大众操作 | 高买低卖 |
| FOMO (错失恐惧) | 追涨杀跌 |
| 泡沫破裂时接盘 | 严重亏损 |

**对策**: 独立思考，按计划执行

## 正确心态

### 1. 概率思维

```
不是每笔交易都赚钱
而是总体盈利概率 > 50%
```

- 接受亏损是成本
- 关注盈亏比
- 坚持系统化交易

### 2. 纪律至上

| 纪律 | 说明 |
|------|------|
| 止损 | 亏损到X%必须走 |
| 仓位 | 单笔不超总资金X% |
| 频率 | 每天最多交易X次 |

### 3. 情绪管理

| 情绪 | 对策 |
|------|------|
| 焦虑 | 降低仓位 |
| 恐惧 | 设好止损 |
| 贪婪 | 止盈策略 |
| 后悔 | 接受不完美 |

---

## 实战心理技巧

### 1. 交易前
- 制定交易计划
- 设定止盈止损
- 记录交易理由

### 2. 交易中
- 不看盘太频繁
- 不临时修改计划
- 专注系统信号

### 3. 交易后
- 记录每笔交易
- 定期复盘总结
- 客观评估表现

---

## 让我变强的心理训练

1. **复盘习惯**: 每天/每周复盘
2. **交易日志**: 记录买卖理由
3. **系统化**: 减少主观判断
4. **仓位管理**: 控制风险敞口

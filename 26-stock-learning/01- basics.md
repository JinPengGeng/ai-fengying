# A股投资入门指南

## 什么是A股？

**A股**是中国境内上市的股票，以人民币计价，主要在 上海证券交易所 和 深圳证券交易所 交易。

---

## 核心概念

### 1. 股票基本概念
| 概念 | 解释 |
|------|------|
| **股价** | 股票的价格 |
| **市盈率 (PE)** | 股价/每股收益，越低越便宜 |
| **市净率 (PB)** | 股价/每股净资产 |
| **股息率** | 股息/股价 |
| **涨跌幅** | 股票每日波动限制（±10%） |

### 2. 交易所
| 交易所 | 代码 | 特点 |
|--------|------|------|
| **上交所** | 600/601/603 | 大盘蓝筹 |
| **深交所** | 000/001/002 | 中小创 |
| **创业板** | 300 | 创新企业 |
| **科创板** | 688 | 科技创新 |

### 3. 指数
| 指数 | 代码 | 代表 |
|------|------|------|
| **上证指数** | 000001.SH | 上交所全部股票 |
| **深证成指** | 399001.SZ | 深交所500家 |
| **创业板指** | 399006.SZ | 创业板100家 |
| **沪深300** | 000300.SH | 300家龙头企业 |

## 常用指标

### 基本面分析
| 指标 | 用途 | 理想值 |
|------|------|--------|
| PE (市盈率) | 估值 | 10-30 |
| PB (市净率) | 估值 | 1-5 |
| ROE | 盈利能力 | >15% |
| 净利润增长率 | 成长性 | >10% |
| 毛利率 | 竞争力 | >30% |

### 技术面分析
| 指标 | 用途 |
|------|------|
| **K线** | 价格走势 |
| **均线** | 趋势判断 |
| **MACD** | 趋势/动量 |
| **KDJ** | 超买超卖 |
| **成交量** | 资金进出 |

## 交易规则

| 规则 | 说明 |
|------|------|
| **交易时间** | 9:30-11:30, 13:00-15:00 |
| **涨跌幅** | ±10%（ST股±5%） |
| **最小交易单位** | 100股（1手） |
| **T+1** | 当天买入，次日才能卖出 |
| **印花税** | 卖出时收0.05% |

---

## 投资策略

### 1. 价值投资
- 寻找被低估的优质公司
- 长期持有
- 关注基本面

### 2. 趋势投资
- 顺势而为
- 技术分析辅助
- 严格止损

### 3. 量化投资
- 利用数学模型
- 程序化交易
- 数据驱动

---

## 风险提示

⚠️ 股市有风险，投资需谨慎
- 不要投入全部资金
- 分散投资
- 设置止损
- 保持理性
